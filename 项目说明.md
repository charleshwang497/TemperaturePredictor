# 明日气温预测器项目说明

## 项目概述

这是一个基于神经网络的明日气温预测器，使用历史天气数据来训练模型，实现对第二天气温的预测。项目设计为学习神经网络算法的完整案例，包含数据处理、模型搭建、训练算法和预测功能的完整流程。

## 搭建思路

### 1. 数据获取与处理
**数据来源：**
- 使用 **Meteostat** Python库获取历史天气数据
- 支持全球任意城市的历史天气数据获取
- 数据包含：温度、湿度、气压、风速、天气状况等

**数据特点：**
- 免费且无需API密钥
- 支持10年以上的历史数据
- 提供小时级和日级数据
- 数据质量可靠，适合机器学习

### 2. 特征工程
**输入特征：**
- 历史温度数据（过去N天的温度）
- 时间特征（月份、日期、季节）
- 天气特征（湿度、气压、风速等）
- 周期性特征（星期、节假日标记）

**输出目标：**
- 第二天的平均温度
- 第二天的最高/最低温度

### 3. 神经网络架构

**网络结构：**
```
输入层 → 隐藏层1 → 隐藏层2 → 输出层
   ↓         ↓         ↓         ↓
  多维    128节点    64节点     1-3节点
```

**激活函数：**
- 隐藏层：ReLU（Rectified Linear Unit）
- 输出层：线性激活（回归问题）

**优化策略：**
- 损失函数：均方误差（MSE）
- 优化器：Adam（自适应学习率）
- 正则化：Dropout + L2正则化

### 4. 算法实现

**训练算法：**
1. 数据预处理：标准化、填充缺失值
2. 构建时间序列样本：滑动窗口方法
3. 网络前向传播：计算预测值
4. 损失计算：比较预测值与真实值
5. 反向传播：计算梯度并更新权重
6. 迭代训练：直到收敛或达到最大轮数

**预测算法：**
1. 输入历史数据
2. 通过网络前向传播
3. 输出第二天的温度预测

## 技术栈

- **Python 3.7+**
- **NumPy**：数值计算
- **Pandas**：数据处理
- **TensorFlow/Keras**：神经网络框架
- **Matplotlib**：数据可视化
- **Meteostat**：天气数据获取

## 运行环境要求

- Python 3.7 或更高版本
- 4GB RAM（推荐8GB）
- CPU即可运行（无需GPU）
- 网络连接（获取天气数据）

## 快速开始

1. 安装依赖包
```bash
pip install -r requirements.txt
```

2. 获取天气数据
```bash
python src/data_collection.py
```

3. 训练模型
```bash
python src/train.py
```

4. 进行预测
```bash
python src/predict.py --city "北京"
```
