## 🎯 常用命令速查

### 数据采集
```bash
# 获取示例数据
python -c "from src.data_collection import WeatherDataCollector; collector = WeatherDataCollector(); collector.load_sample_data()"

# 获取真实数据
python -c "from src.data_collection import WeatherDataCollector; collector = WeatherDataCollector(); collector.collect_weather_data('北京', '2020-01-01', '2023-12-31')"
```

### 模型训练
```bash
# 训练LSTM模型
python src/train.py --model_type lstm --epochs 100

# 训练GRU模型
python src/train.py --model_type gru --epochs 100

# 指定批次大小
python src/train.py --model_type lstm --epochs 100 --batch_size 64
```

### 温度预测
```bash
# 预测明天温度
python src/predict.py --model models/temperature_model.h5 --city 北京

# 预测未来3天
python src/predict.py --model models/temperature_model.h5 --city 北京 --prediction_days 3

# 使用最近数据进行预测
python src/predict.py --model models/temperature_model.h5 --city 北京 --use_recent
```

### 可视化
```bash
# 查看训练历史
python -c "from src.visualization import WeatherVisualizer; v = WeatherVisualizer(); import json; h = json.load(open('models/training_history.json')); v.plot_training_history(h)"
```

## 🔧 常见问题解决

### 问题1：找不到模块
**错误**: `ModuleNotFoundError: No module named 'src.xxx'`

**解决**: 
```bash
# 确保在项目根目录下运行
pwd  # 应该显示包含main.py的目录

# 或者添加PYTHONPATH
export PYTHONPATH=$PYTHONPATH:$(pwd)
```

### 问题2：没有数据文件
**错误**: `FileNotFoundError: No such file or directory: 'data/xxx.csv'`

**解决**:
```bash
# 先运行数据采集
python main.py
# 选择 "1. 数据采集与预处理"
```

### 问题3：没有模型文件
**错误**: `No model files found`

**解决**:
```bash
# 先训练模型
python main.py
# 选择 "2. 训练模型"
```

### 问题4：训练太慢
**解决**: 
```bash
# 减少训练轮数
python src/train.py --epochs 50

# 或者使用较小的批次大小
python src/train.py --epochs 100 --batch_size 16
```

### 问题5：内存不足
**解决**:
```bash
# 减小序列长度
python src/train.py --sequence_length 20

# 或者使用较小的批次大小
python src/train.py --batch_size 16
```

## 📊 项目文件说明

### 核心文件
- `main.py` - 主程序入口，提供交互式界面
- `requirements.txt` - Python依赖包列表
- `README.md` - 详细的项目说明文档

### 源代码 (src/)
- `data_collection.py` - 数据收集模块
- `data_preprocessing.py` - 数据预处理模块
- `neural_network.py` - 神经网络模型
- `train.py` - 训练脚本
- `predict.py` - 预测脚本
- `visualization.py` - 可视化工具

### 数据目录 (data/)
- `sample_weather_data.csv` - 示例天气数据
- `processed/` - 预处理后的数据

### 模型目录 (models/)
- `*.h5` - 训练好的模型文件
- `*_config.json` - 模型配置文件
- `*_history.json` - 训练历史

### 预测结果 (predictions/)
- `*.json` - 预测结果文件
- `*.csv` - 预测报告
- `*.png` - 可视化图表

## 💡 实用技巧

### 1. 快速测试
```bash
# 使用较少的训练轮数快速测试
python src/train.py --epochs 10
```

### 2. 批量训练
```bash
# 训练多个模型进行比较
for model in lstm gru; do
    python src/train.py --model_type $model --epochs 100
done
```

### 3. 自动化预测
```bash
# 每天自动预测
#!/bin/bash
python src/predict.py --model models/latest_model.h5 --city 北京
crontab -e
# 添加: 0 8 * * * /path/to/predict.sh
```

### 4. 结果导出
```python
# 将预测结果导出为Excel
import pandas as pd
import json

with open('predictions/prediction.json', 'r') as f:
    data = json.load(f)

df = pd.DataFrame(data['predictions'])
df.to_excel('prediction_report.xlsx', index=False)
```
